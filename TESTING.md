# ClinicCore 2.0 — Client Testing Guide

Version: 0.1.0 | Last updated: 2026-03-17

## How to use this guide

This guide is written for non-technical testers. Each section describes a feature, how to test it step by step, and what you should expect to see. No coding knowledge is needed.

If something does not work as described, note down:
1. What you were trying to do
2. What you expected to happen
3. What actually happened (include any error messages)

## Reporting Issues

Please report any issues to the project team via Jira or directly in the team communication channel.

Include:
- Which feature you were testing
- The steps you took
- What you expected vs. what happened
- A screenshot if possible
